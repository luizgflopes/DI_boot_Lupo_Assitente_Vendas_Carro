# DI_boot_Lupo_Assitente_Vendas_Carro
   Prompt estruturado para vendedores de carros ou chatbots. Conduz o cliente pergunta por pergunta, coleta até 7 informações principais e sugere até 3 veículos adequados. Inclui validação, refinamento com perguntas extras e recomenda carros similares quando há marca ou modelo de interesse.
